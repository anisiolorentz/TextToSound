Tutorial

How to use the text to music converter:

  • Letters A-G (a-g): Music notes
  • Space: Silence
  • + / -: Increase/Descrease Volume
  • R+ / R-: Increase/Decrease the current octave
  • BPM+ / BPM-: Increase/Decrease BPM
  • Enter (new line): Changes instrument to piano
  • "?": Plays a random note
  • I, O, U (or i, o, u): Plays last note or ringing sound
  • ";": Sets a random BPM value

Example "abcR+defBPM+gg dc"
